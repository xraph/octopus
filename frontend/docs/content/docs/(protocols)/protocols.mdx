## Protocol Detection

Octopus automatically detects protocols based on:

- HTTP headers (`Upgrade`, `Content-Type`)
- URL path patterns
- Port configuration
- Request method

## Configuration

```yaml
routes:
  # HTTP/REST
  - path: /api/*
    upstream: api-service
    protocol: http
  
  # gRPC
  - path: /grpc.*
    upstream: grpc-service
    protocol: grpc
  
  # WebSocket
  - path: /ws
    upstream: ws-service
    protocol: websocket
  
  # SSE
  - path: /events
    upstream: event-service
    protocol: sse
```
