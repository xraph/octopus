## Admin API Endpoints

### Health

```bash
GET /_/health
```

Returns gateway health status.

### Metrics

```bash
GET /metrics
```

Returns Prometheus metrics.

### Services

```bash
GET /_/services
```

Lists discovered services.

### Routes

```bash
GET /_/routes
```

Lists active routes.

### Configuration

```bash
GET /_/config
```

Returns current configuration.

### Reload

```bash
POST /_/reload
```

Reloads configuration.
