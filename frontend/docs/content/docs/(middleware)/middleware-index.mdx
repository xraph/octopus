## Configuration

```yaml
middleware:
  - auth
  - rate_limit
  - cors
  - compression

auth:
  type: jwt
  secret: ${JWT_SECRET}

rate_limit:
  requests_per_second: 100
  burst: 200

cors:
  allowed_origins: ["*"]
  allowed_methods: [GET, POST, PUT, DELETE]

compression:
  enabled: true
  level: 6
```

## Execution Order

Middleware executes in the order specified:

```
Request → Auth → Rate Limit → CORS → Upstream → CORS → Compression → Response
```
