## Using Plugins

```yaml
plugins:
  - name: jwt-auth
    builtin: true
    config:
      secret: ${JWT_SECRET}
      algorithm: HS256
  
  - name: redis-cache
    path: /plugins/redis-cache.so
    config:
      url: redis://localhost:6379
      ttl: 300
```
