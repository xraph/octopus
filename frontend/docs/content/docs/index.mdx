## Quick Example

```yaml
# config.yaml
server:
  http_port: 8080
  admin_port: 9090

discovery:
  backend: kubernetes
  kubernetes:
    namespace: default

farp:
  enabled: true
  watch_interval: 5s
```

```bash
# Run the gateway
octopus --config config.yaml

# Check health
curl http://localhost:9090/_/health

# View metrics
curl http://localhost:9090/metrics
```

## Getting Started

<Steps>

<Step>
### Install Octopus

```bash
# From source
cargo install octopus-gateway

# Or download binary
curl -L https://github.com/xraph/octopus/releases/latest/download/octopus-linux-amd64 -o octopus
chmod +x octopus
```
</Step>

<Step>
### Create Configuration

```yaml
# config.yaml
server:
  http_port: 8080
  admin_port: 9090

discovery:
  backend: static

upstreams:
  - name: my-service
    instances:
      - address: localhost
        port: 8081

routes:
  - path: /api/*
    upstream: my-service
```
</Step>

<Step>
### Run the Gateway

```bash
octopus --config config.yaml
```
</Step>

<Step>
### Test It

```bash
# Make a request
curl http://localhost:8080/api/hello

# Check health
curl http://localhost:9090/_/health

# View metrics
curl http://localhost:9090/metrics
```
</Step>

</Steps>
