# Guides

Each guide is an end-to-end walkthrough for a single task, with copy-pasteable
configuration grounded in the real schema and links to the reference pages. If
you are brand new, start with [Quick start](/docs/quick-start) and
[Your first gateway](/docs/first-gateway), then come back here for specific
tasks.

<Callout type="info">
  Every guide below only relies on features that are wired into the runtime
  today. Where a related feature is parsed but not yet enforced (for example
  rate limiting), the guide says so explicitly rather than pretending it works.
</Callout>
